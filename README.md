## 🚀 ¿Cómo usar este proyecto?

1. Lee cada archivo `.md` en orden
2. Haz los ejercicios en las carpetas `practica_XX`
3. Usa el **truco de memoria** al final de cada sección
4. El archivo `GLOSARIO.md` es tu diccionario siempre disponible
5. `CHEATSHEET.md` = tu hoja de trucos rápida

## 💡 Truco MAESTRO para recordar todo el ecosistema

**"El Chef con Herramientas en un Restaurante Automatizado"**

- 🧑‍🍳 **El Chef** = El Agente de IA (toma decisiones)
- 🔧 **Las Herramientas** = Los MCP Servers (acceso a datos/acciones)
- 🗣️ **El Protocolo** = MCP (el idioma entre chef y herramientas)
- 🤖 **El Robot Ayudante** = n8n (automatiza tareas repetitivas)
- 🏠 **La Cocina Propia** = IA Local (sin depender de internet)
